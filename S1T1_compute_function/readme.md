		* `test_elementaries()` - проверит погрешности расчёта элементарных функций

		* `is_elementary()` - проверяет, что f(x) - элементарная (уже реализована)
		* `estimate_abs()` - оценка модуля f(x) в окрестности x0 (уже реализована)
